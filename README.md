# JS Assignment (Fundamentals of Web Design)

## 📌 Description
This repository contains solutions to Assignment 4 using JavaScript.  
All inputs are taken using prompt() and outputs are shown using alert().

---

## 📂 Structure
- Q1.js → Digit Gatekeeper
- Q2.js → Roll-Seed Lock
- Q3.js → Mirror Corridor
- Q4.js → Fare Calculator
- Q5.js → Skipping Numbers
- Q6.js → Contest Score Judge
- index.html → Run all programs

---

## ⚙️ How to Run
1. Open index.html in browser
2. Click any button (Q1–Q6)
3. Enter inputs in prompt
4. See result in alert

---

## 🧠 Approach

### Q1
Loop through range, check divisibility, digit sum prime, and no zero.

### Q2
Apply transformation rules 3 times, then validate conditions.

### Q3
Check for smallest X such that N+X is palindrome and divisible by K.

### Q4
Step-by-step fare calculation using conditions and rounding.

### Q5
Add numbers skipping multiples of (seed+2) until sum ≥ N.

### Q6
Compute score and apply rules for final status.

---

## ⏱️ Complexity
- Most problems: O(N) or O(N log N)
- Efficient for given constraints
